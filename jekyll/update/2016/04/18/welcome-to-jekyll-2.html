Applescript小试水

Apr 18, 2016

每天下班的时候，会手动关闭一系列办公软件，然后被电脑回家，而自己再接触mac后，也放弃了关机的习惯。因此第二天上班都需要手动打开一系列软件，于是很想试试有没有办法写一个小脚本，让这些需要的程序自动运行。

浏览了很多网页后，使用Applescript写出了满足自己需求的小程序。通过Spotlight键入Applescript打开编辑器，使用tell命令完成需求。 代码如下:

tell application "Slack"
	activate
end tell

tell application "Mail"
	activate
end tell

tell application "Calendar"
	activate
end tell

tell application "Google Chrome"
	activate
	open location "https://xxxx.com"
end tell

最后导出为程序，保存为 GoToWork.app。

自定义app图标:

  • 制作一个icns图像文件，可以自己制作png格式，然后通过在线转换器转换为icns格式。命名为 applets.icns。

  • 右击（两指）GoToWork.app展示内容，替换 Contents/Resources 中的applets.icns文件。

一个小程序，就这样完成啦！！🎉🎉🎉
